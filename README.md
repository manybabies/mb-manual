# ManyBabies Manual
## Policies, guidelines, and resources for the ManyBabies Consortium


 
Available here: https://manybabies.org/mb-manual

Manual is under development (as of November, 2025). 

To build locally (this is uncommon):
* Make sure Quarto is up to date (we use HomeBrew)
* Restore dependencies using `renv::restore()`
